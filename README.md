# simplecrudjs
Kuliah WebProgramming: CRUD sederhana 26

## Persiapan
1. Install Node JS
2. Pastikan Database dan table tersedia, sesuai dengan server.js
3. Lakukan installasi nodemon, express, mysql2, cors, body-parser
```bash
npm i -g nodemon

npm i express mysql2 cors body-parser
```
4. Untuk menjalankan server
```bash
nodemon server
```
